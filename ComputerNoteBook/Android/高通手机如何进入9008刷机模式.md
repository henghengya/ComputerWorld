# 高通手机如何进入9008模式

## 命令方式

1. 打开USB调试，使用ADB，这需要ADB驱动，执行命令：
	```bash
	adb reboot edl
	```

2. 进入手机Fastboot模式，这需要fastboot驱动，执行命令：

	``` bash
	fastboot oem edl
	```
